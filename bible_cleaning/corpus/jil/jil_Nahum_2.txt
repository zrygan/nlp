1: Ninive, ginasalakay ikaw!
: Ang gahom nga magalaglag sa imo nag-abot na.
: Butangi sang mga soldado ang mga pamakod!
: Bantayi ang mga dalan!
: Tipuna sing tingob ang inyo mga soldado kag maghanda sa pag-away!
2: Bisan nagpangagaw sang mga pagkabutang ang mga kaaway kag nagpanglaglag sang talamnan sang ubas, ang Ginoo magauli sang dungog kag gahom sang Juda, kasubong sang iya ginsaad sa pagpabalik sang dungog kag gahom sang Israel.
3: Ang kaaway nga mga soldado nagadala sing pula nga mga taming
: kag nagabayo sing pula.
: Nagahanda sila sa pagsalakay!
: Ang ila mga karwahi nagakirab kaangay sang kalayo!
: Ang ila mga kabayo nagaaliwasa!
4: Ang mga karwahi nagalabad sa mga dalan,
: nagapakadto pakari sa mga plasa.
: Nagaidlab sila kaangay sang sulo,
: nagakirab kaangay sang kilat.
5: Ang mga opisyales ginpatawag,
: nagapalanandad sila sa ila pagpalapit.
: Ang mga manugsalakay nagadalidali pakadto sa pader,
: kag ginbutang ang panagang para sa inugbungkal sang pader.
6: Ang mga pwerta nga ginaagyan sang sa suba nabuksan,
: ang palasyo napun-an sing kahadlok.
7: Ang rayna nadakpan,
: ang iya mga ulipon nagaugayong kaangay sang mga pating kag nagahibi,
: ginapukpok nila ang ila mga dughan sa kasubo.
8: Kaangay sang tubig sa nabulwang nga punong
: ang mga tawo nagadinaguso halin sa Ninive!
: Dulog! Dulog! mabatian ang singgit,
: pero wala sing may nagbalikid.
9: Panguhaa ang pilak!
: Panguhaa ang bulawan!
: Ang siudad puno sang manggad!
10: Ang Ninive nalaglag, napatumbayaan, mamingaw!
: Ang mga tagipusoon nagapalanguyos sa kahadlok,
: ang mga tuhod nagakurog, ang kusog wala na,
: ang mga guya nagapalanglapsi.
11: Diin na karon ang siudad nga kaangay sang lungib sang mga leon,
: ang lugar sa diin ang mga tutoy nga leon nagatiti,
: sa diin ang inang nga leon kag ang iya mga tutoy
: nagakadto kag hilway sa katalagman?
12: Ang leon nagapatay sang iya biktima,
: ginakusnit niya ini para sa iya pares kag para sa mga tutoy sini,
: ginapuno niya ang iya lungib sang kinusnit nga unod.
13: Ang Ginoo sang mga hangaway nagasiling, Ako ang imo kaaway. Sunogon ko ang imo mga karwahi. Ang imo mga soldado mapatay sa inaway, kag kuhaon ko ang tanan nga ginpanguha mo sa iban. Ang mga pagpangayo sang imo mga mensahero sa iban nga mga pungsod indi na gid mabatian.
