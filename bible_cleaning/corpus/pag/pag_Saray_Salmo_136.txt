1: Misalamat kayod Katawan , ta sikato et maung tan andi-anggaan so panangaro to.
2: Misalamat kayod sankatalonggaringan ed saray diriyos;
: andi-anggaan so panangaro to.
3: Misalamat kayod sankamapanyarian ed saray kakatawan;
: andi-anggaan so panangaro to.
4: Sikato labat so manggagawad saray ankakabaleg a milagro;
: andi-anggaan so panangaro to.
5: Diad kakabatan to, pinalsa to ray katawenan;
: andi-anggaan so panangaro to;
6: impasen toy dalin ed tapew na saray danum;
: andi-anggaan so panangaro to.
7: Pinalsa toy agew tan say bulan;
: andi-anggaan so panangaro to;
8: say agew a manoley no agew;
: andi-anggaan so panangaro to;
9: say bulan tan saray bitewen a manoley ira no labi;
: andi-anggaan so panangaro to.
10: Pinatey to ray lalakin pangulowan ya anak na saray Egipsio;
: andi-anggaan so panangaro to.
11: Impaway tod Egipto iray totood Israel;
: andi-anggaan so panangaro to;
12: diad panamegley na mabiskeg tan makapanyari ran taklay to;
: andi-anggaan so panangaro to.
13: Binelsay toy Dayat ya Ambalanga;
: andi-anggaan so panangaro to.
14: Pinadalan tod pegley na dayat iray totoo to;
: andi-anggaan so panangaro to;
15: balet impalner toy ari na Egipto tan saray sondalo to;
: andi-anggaan so panangaro to.
16: Inwanwan to ray totoo to diad imbeltang dad kalawakan;
: andi-anggaan so panangaro to.
17: Pinapatey to ray makapanyarin arari;
: andi-anggaan so panangaro to;
18: pinapatey to ray bantug ya arari;
: andi-anggaan so panangaro to;
19: si Sehon ya ari na saray Amorreo;
: andi-anggaan so panangaro to;
20: tan si Og ya ari na Basan;
: andi-anggaan so panangaro to.
21: Inter to ray dalin da ed saray totoo to;
: andi-anggaan so panangaro to;
22: inter to rad Israel ya aripen to;
: andi-anggaan so panangaro to.
23: Ag to itayo nilingwanan nen atalo itayo;
: andi-anggaan so panangaro to;
24: inekal to itayod pakayari na saray kalaban tayo;
: andi-anggaan so panangaro to.
25: Iitdan toy tagano iray amin a pinalsan mabilay;
: andi-anggaan so panangaro to.
26: Misalamat kayod Dios na tawen;
: andi-anggaan so panangaro to.
