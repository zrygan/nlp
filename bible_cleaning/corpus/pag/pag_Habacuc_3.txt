1: Sikato yay pikakasi nen propita Habacuc
2: O Katawan , nadngelan ko so ginagawam,
: et mapalaloy pankelaw ko.
: Diad sayan panaon mi gawam pa natan
: ira may makapakelaw a ginagawam nen saman.
: Mangasi ka pa, anggan mamapasnuk ka.
3: Onsasabi lamet a manlapud Edom so Dios;
: onsasabin manlapud kapokdulan na Paran so masanton Dios.
: Akobutan na kagalangan to so katawenan,
: tan napno na dayew to so sankamundoan.
4: Onsasabin maliwliwawa a singa kirmat;
: mankikidyam ed lima to so silew,
: a diay akaiyamutan na pakayari to.
5: Papaonaen tod arap to so salot
: tan gangganen ton ontombuk ed sikato so patey.
6: No sikatoy ontonda, nayegyeg so mundo;
: diad ilinggis to onggiwgiw iray bansa.
: Abuybuyak iray andi-anggaan a kapalandeyan;
: linmereg iray ag manganggan kapokdulan,
: saray kapokdulan a nanaakaran to ra nen saman.
7: Anengneng kon antakot iray totood Cusan
: tan manggiwgiw iray totood Madian.
8: Saray ilog so amapasnuk ed sika, Katawan ?
: Say dayat so amapetang na olom?
: Linmogan kad tapew na kaloreman;
: karwahim so lorem ya ipapalir na bagyo,
: nen inyakar moy pananalo ed saray totoom.
9: Imparaan mon ikana so panam;
: akaparaan kan mangibolus na balay panam.
: Papalduaen na kirmat mo so mundo.
10: Nen anengneng ka na kapalandeyan ginmiwgiw ira;
: naibung so danum manlapud saray tawen.
: Inmungol iray danum ed dalem na dalin,
: tan binmabaleg iray daluyon da.
11: Lapud kidyam na maples iran panam
: tan say siayap na makasileng a pikam,
: nantanyareg so agew tan say bulan.
12: Sipapasnuk kan binmeltang ed mundo;
: diad gaway sanuk mo ginagatinan mo ray bansa.
13: Pinmaway kan mangilaban ed saray totoom,
: tan mangiliktar ed pinilim ya ari.
: Kinabil moy pangulo da ray totoon maoges
: tan sigput mon dineral iray totoo to.
14: Tinoruk na saray panam so pangulo da ray sondalo to,
: nen sinmabi ran singa bagyo ya angikayat ed sikami,
: malikeliket iran singa saray totoon maamut a mamapairap ed saray mairap-bilay.
15: Impagatin-gatinan mod saray kabayom so dayat,
: et binmora-bora so mabiskeg a danum.
16: Narerengel ko ra yan amin, et manggiwgiw ak;
: manggigiwgiw ed takot iray bibil ko.
: Paligpalig so laman ko,
: mankitakit iray salik.
: Mandeen ak a manalagar ed isabiy panaon
: a dusaen na Dios iray ondodogpad sikami.
17: Anggan anggapoy bungay igos
: tan saray kaubasan,
: anggan anggapoy naborbor ed kaolibaan
: tan anggapoy naanid kaalogan,
: anggan ompatey ya amin iray karniro
: tan naandian na baka iray koral,
18: manliket tan manggayaga ak nin siansia,
: lapud say Katawan a Dios so manangilaban ko.
19: Pinabiskeg ak na Katawan a Dios .
: Patalabiten to ray salik a singa sali na ulsa
: tan iyarawi to ak ed kaatapan diad kapalandeyan.
