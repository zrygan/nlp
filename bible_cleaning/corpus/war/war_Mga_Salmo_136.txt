1: Adton Ginoo pagpasalamatan ta,
: kay padayon gud an iya paghigugma.
: Pagpasalamatan kay maupay hiya,
: kay padayon gud an iya paghigugma.
2: Dyos han kadiyosan pasalamatan ta
: kay padayon gud an iya paghigugma.
: Labaw han tanan nga kadiyosan hiya,
: kay padayon gud an iya paghigugma.
3: An gigagamhani pasalamatan ta,
: kay padayon gud an iya paghigugma;
: Ginoo han mga Ginoo man hiya,
: kay padayon gud an iya paghigugma.
4: An nagbubuhat hin dagko hiya gudla,
: kay padayon gud an iya paghigugma.
: Dagko'ng mga milagro an buhat niya,
: kay padayon gud an iya paghigugma.
5: An kalangitan amo an buhat niya,
: kay padayon gud an iya paghigugma.
: Iya ginbuhat han kinaadman niya,
: kay padayon gud an iya paghigugma.
6: Amo hiya an nagbitad hadton tuna,
: kay padayon gud an iya paghigugma.
: Ha bawbaw han katubigan gintindog n'ya,
: kay padayon gud an iya paghigugma.
7: Iya ginhimo an dagko nga kapawa,
: kay padayon gud an iya paghigugma.
: An adlaw ngan an bulan ginhimo niya,
: kay padayon gud an iya paghigugma.
8: An adlaw hin paghadi an kaadlawon,
: kay padayon gud an iya paghigugma.
9: Bulan, kabiton-an hadi'n kagab-ihon,
: kay padayon gud an iya paghigugma.
10: Mga Ehiptohanon ginpatay niya,
: kay padayon gud an iya paghigugma.
: Mga suhag nga kalalaken-an nira,
: kay padayon gud an iya paghigugma.
11: Tikang ha Ehipto ginmandoan hira,
: kay padayon gud an iya paghigugma.
: An katawhan ini han Israel niya,
: kay padayon gud an iya paghigugma.
12: Pinaagi han kusgan nga kamot niya,
: kay padayon gud an iya paghigugma.
: Han gamhanan nga butkon gindara hira,
: kay padayon gud an iya paghigugma.
13: Ginbuhag niya adton Dagat Pula,
: kay padayon gud an iya paghigugma.
14: Dida hin ginpaagi'n katawhan n'ya,
: kay padayon gud an iya paghigugma.
15: An hadi han Ehipto ginlumos niya,
: kay padayon gud an iya paghigugma.
: Pati an mga kasondalohan niya,
: kay padayon gud an iya paghigugma.
16: An iya katawhan ginmandoan niya,
: kay padayon gud an iya paghigugma.
: Paglatas han kamingawan nga mamara,
: kay padayon gud an iya paghigugma.
17: Gamhanan kahadian ginpatay niya,
: kay padayon gud an iya paghigugma.
18: Bantog nga kahadian ginpatay niya,
: kay padayon gud an iya paghigugma.
19: Ginpatay hadi Sihon nga Amorita,
: kay padayon gud an iya paghigugma.
20: Pati hi Og nga hadi han Basan hiya,
: kay padayon gud an iya paghigugma.
21: Iya iginhatag adton ira tuna,
: kay padayon gud an iya paghigugma.
: Ginhatag ngadto hadton katawhan niya,
: kay padayon gud an iya paghigugma.
22: Ngadto ha Israel iginhatag niya,
: kay padayon gud an iya paghigugma.
: Ha kan Israel nga surugoon niya,
: kay padayon gud an iya paghigugma.
23: Waray gudman kita niya paglimota,
: kay padayon gud an iya paghigugma.
: Bisan dida han ginpirde na gud kita,
: kay padayon gud an iya paghigugma.
24: Ngan iya man nga ginpabuhian kita,
: kay padayon gud an iya paghigugma.
: Ginlibre tikang han mga kaaway ta,
: kay padayon gud an iya paghigugma.
25: Hin mga pagkaon gintatagan niya,
: kay padayon gud an iya paghigugma.
: Tanan mga buhi nga binuhat niya,
: kay padayon gud an iya paghigugma.
26: Ighatag an mga pagpasalamat ta,
: kay padayon gud an iya paghigugma.
: An Dyos han langit pagpasalamati na,
: kay padayon gud an iya paghigugma.
